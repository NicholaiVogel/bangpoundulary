# Rectumultuous

adj. When the back end is going through it. Gastrointestinal turbulence. The colon has entered a state of unrest. Wombo Law #1: "Rectum" intact (surrenders isolation), "Tumultuous" → "umultuous" (surrenders "T-"). Asymmetrical sacrifice but medically precise. Valid. Previously misdefined as a treatise on traumatic memory due to an etymological error by the scribe. The scribe has been walnutted.
